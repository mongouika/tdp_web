timeline

to > to > to ....
メソッドチェーン化してまとめたものをtimelineとして管理します 

            
                gsap.set(".square", { opacity: 0, x: 100 ,color: red});//初期状態の設定。

                const tl = gsap.timeline({ repeat: -1, repeatDelay: 1 });
                
                tl.to(".num1", {
                  opacity: 1, x: 0,color:red
                }).to(".num2", {
                  opacity: 1, x: 0
                }).to(".num3", {
                  opacity: 1, x: 0
                }).to(".num4", {
                  opacity: 1, x: 0
                }).to(".num1", {
                  opacity: 0, x: -100
                }).to(".num2", {
                  opacity: 0, x: -100
                }).to(".num3", {
                  opacity: 0, x: -100
                }).to(".num4", {
                  opacity: 0, x: -100
                })
